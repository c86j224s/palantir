# 👁️ Palantir
> **Minimalist Kubernetes Observer & Manager**

Built with **Rust** & **React**. Decoupled core logic, high-density telemetry, and interactive control.

---

## 🚀 Quick Start

### 1. System Requirements (Linux/WSL2)
```bash
sudo apt update && sudo apt install -y build-essential curl wget file libssl-dev libgtk-3-dev libayatana-appindicator3-dev librsvg2-dev libwebkit2gtk-4.0-dev libsoup2.4-dev
```

### 2. Environment Setup
```bash
npm install
npm install --prefix frontend
```

### 3. Execution
```bash
npm run dev
```

## 🛠️ Tech Stack
- **Core Engine**: Rust (`kube-rs`, `tokio`)
- **Bridge**: Tauri
- **Interface**: React, Tailwind CSS, Framer Motion, xterm.js
